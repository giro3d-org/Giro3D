--- title: Multi-resolution elevation data shortdesc: Combine multiple elevation sources into a single layer. longdesc: The AggregateImageSource can combine multiple ImageSources into one. One the many benefits of this approach is that is reduces the number of layers in a map, which helps improve performance.
