--- title: Interactive vector layer shortdesc: Display interactive vector data on a Map. attribution: © OpenLayers longdesc: The VectorSource accepts all formats that OpenLayers handle (such as GeoJSON, KML, GPX...). The VectorSource accepts either a URL to the remote data file, or the text content of the file, or an array of OpenLayers features. The style of the source is expressed using the OpenLayers Style object. tags: [map, layer, vector, geojson] ---
